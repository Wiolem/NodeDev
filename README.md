# 一款简单的 Express 构建的职位管理后端项目

## 组件化/MVC
- 前端界面利用jQuery采用组件化思想实现界面显示
- 后端采用MVC的设计典范

## 功能实现
- 登陆/注册
- 持久登陆
- 列表展示
- 添加/修改/删除职位
- 图片上传
- 筛选职位
